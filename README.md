# K-Tools

**K-Tools** — это минималистичное desktop-приложение для быстрой обработки медиафайлов, вдохновлённое Microsoft PowerToys. Написано на Python с использованием [PyQt6-Fluent-Widgets](https://github.com/zhiyiYo/PyQt-Fluent-Widgets).

## ✨ Возможности

### 1. 🧹 Очистка метаданных
Удаляет все метаданные из видеофайлов, сохраняя оригинальные потоки (copy).
*   **Использование**: Перетащите файлы, выберите суффикс (по умолчанию `_cl`) и нажмите «Выполнить».
*   **Опции**:
    *   Суффикс выходного файла.
    *   Удаление исходного файла после успешной обработки.

### 2. 🔄 Конвертация контейнеров
Сменяет формат контейнера без перекодирования видео и аудио потоков (`-c copy`). Мгновенная конвертация.
*   **Поддерживаемые форматы**: MKV ↔ MP4 ↔ MOV ↔ AVI.
*   **Опции**:
    *   Выбор целевого формата (MP4/MKV...).
    *   Удаление исходного файла.

### 3. 🎵 Универсальный Аудио Конвертер
Конвертирует аудиофайлы в любые популярные форматы с настройкой качества.
*   **Поддерживаемые форматы**:
    *   MP3, AAC, OGG, WMA, OPUS (с выбором битрейта).
    *   FLAC, WavPack (с выбором уровня сжатия).
    *   AC3, EAC3, DTS (для домашнего кинотеатра).
    *   WAV, AIFF, ALAC, ADPCM (PCM форматы).
*   **Особенности**:
    *   Сохраняет «чистые» потоки (**Raw Streams**) без лишних метаданных контейнера (`-map_metadata -1`).
    *   AAC сохраняется как `.aac` (ADTS), а не `.m4a`.
    *   AC3/DTS/EAC3 сохраняются как `.ac3`/`.dts`/`.eac3`.
*   **Опции**:
    *   Целевой формат.
    *   Битрейт (для lossy) или степень сжатия (для lossless).
    *   Удаление исходного файла.

### 4. 🎞️ Муксер (Video + Audio + Subs)
Собирает MKV-файл из разрозненных компонентов (видео, аудио, субтитры).
*   **Логика**: Автоматически группирует файлы с одинаковым именем (например, `Movie.mp4` + `Movie.ac3` + `Movie.srt`).
*   **Инструмент**: Использует `mkvmerge` (MKVToolNix) для надежной сборки без перекодирования.
*   **Опции**:
    *   Заголовок для дорожки субтитров (например, "Russian").
    *   Удаление "лишних" дорожек из исходного видео (оставляет только видео-поток + новые аудио/сабы).
    *   Удаление "лишних" дорожек из исходного видео (оставляет только видео-поток + новые аудио/сабы).
*   **Интерфейс**: Специальная таблица с "умной" сортировкой (Natural Sort) для правильного порядка эпизодов.

## 🛠️ Установка и запуск

### Требования
*   Windows 10/11
*   Python 3.10+
*   FFmpeg (должен быть в PATH или рядом с приложением)
*   MKVToolNix (для работы Муксера)

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск
```bash
python main.py
```

## 🏗️ Структура проекта
*   `app/core` — базовые классы и реестр скриптов.
*   `app/scripts` — реализации скриптов (плагины).
*   `app/ui` — интерфейс на базе Fluent Widgets.
*   `app/infrastructure` — обертка над FFmpeg.
